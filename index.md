# Scripts Repository

This repository contains various scripts that can be easily pulled and used.

## Available Scripts

### opam_setup.sh
Setup script for OCaml and OPAM package manager.

**Usage:**
```bash
curl -fsSL https://durbatuluk1701.github.io/scripts/opam_setup.sh | bash
```

Or download and run:
```bash
wget https://durbatuluk1701.github.io/scripts/opam_setup.sh
chmod +x opam_setup.sh
./opam_setup.sh
```

**What it does:**
- Installs bubblewrap and libgmp-dev dependencies
- Installs OPAM (OCaml package manager)
- Initializes OPAM
- Sets up the OPAM environment
